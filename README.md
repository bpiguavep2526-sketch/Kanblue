## DESCRIPCIÓN DEL PROYECTO

### ¿QUE ÉS KANBLUE?

Kanblue es nuestra propuesta para gestión de proyectos y tareas. Consta de una aplicación que permite crear proyectos, los cuales contienen tareas adjuntas con características personalizables como asignación,
título y descripción, estado...
Siguiendo la metodología KANBAN, dispondrá de un tablón de 4 columnas para poder gestionar y llevar un control sobre las tareas.

Nuestro objetivo con Kanblue, es proporcionar una interfaz simple y clara para que equipos experimentados y novatos puedan gestionarse sus proyectos de forma rápida y eficiente, con pocas complicaciones.

***

## FUNCIONES DE LA APLICACIÓN

La aplicación permite:
- Registro de Usuarios
- Inicio de sesión
- Ver, crear y editar proyectos (disponibles en una vista de lista)
- Interactuar con tareas (ver, editar, crear, arrastrar entre columnas, ver gráficos...)
- Editar tu información de usuario.

***

## CARACTERÍSTICAS

La aplicación ha sido creada usando HTML, JS, CSS y Bootstrap y Laravel.
Utiliza la libreria [Charts.js](https://www.chartjs.org/docs/latest/) para crear los gráficos.
